Overview

The module allows users to register to their Odoo account using One-time Password via Twilio SMS. So, users don't need to bother typing email ids every time, they can get it done through a single OTP verification process.

Features

Sign-up/Register through Mobile Number via OTP verification.

Sign-in with mobile number

OTP Verification for"Reset Password".

Automatic IP detection for Mobile Number field.

Select Country code manually.

Field validation for Mobile Number field.

Timer for OTP verification.

Setup & Configuration

Prerequisites

  • Python Library Installation

    To install this module, you need to have installed

  • Ubuntu

    sudo pip install twilio

Configuration

When one install's the module the necessary configuration will automatically appear in the system:

  • Free Account(b2c) will be enabled in Settings>General Settings> Customer Account
  • Employee rights will be assigned if user is of type Portal.

    • Twilio SMS Configuration

  • Enter your Twilio Account credentials

    • Screenshots

    Signup/Register with Mobile Number

  • Instead of entering email id and password, user can save their time by registering to the account via mobile number.
  • Once you have installed this module, you can find a "Sign-up with Mobile Number" link under the Sign-Up button.

    • Automatic IP Detection .

  • As a next step, the module will automatically detect system IP and set country code accordingly for the mobile number field. So, all you need to do is enter your mobile number, name, and password and you are all set with your account.

    • Manually Select your Country Code:

  • User can change country code manually from the given dropdown list

    • Mobile Number Validation:

  • A field validation is applied to check if user hasn't entered any invalid mobile number referring to the country code.
    • #country_code.gif
  • Signup button is disabled if user enters an invalid mobile number:
  • Signup button is enabled if user enters a valid mobile number. :

    • OTP Verification Process :

  • An OTP verification code will be sent on the registered Mobile Number.
  • Upon verifying OTP, system will create User account according to the Mobile Number and other field values.

    • OTP Validation:

  • You can check your mobile device for OTP and enter it in the "Verify OTP" field.

    • Timer For Resend OTP .

  • As per standard verification process, the OTP lasts for few seconds. Therefore, a timer is shown on the OTP verification page. If the timer is completed, a "Resend OTP" link will be enabled to help user make another attempt for verification.

    • Login With Mobile Link :

    If user already has an account created in Odoo with Mobile number, then he/she can login with Mobile number as well.

  • User can find a "login with mobile number" link on Login page Under the "Login" button.

    • Login With Mobile Number:

  • User can enter their registered mobile number in the "Mobile" field to initiate the login process.
  • Enter your account password along with mobile number to complete the login process.

    • User Account View :

  • As you can see in the below screenshot, system user will be created with provided Values.

    • Reset Password Page :

  • On Reset Password Page you have two options- Reset Password via: Email or Mobile Number.

    • OTP Verification for Password Reset:

  • Upon entering mobile number, an OTP Verification Code will be sent to your registered Mobile.
  • If user enters incorrect OTP Value, system will generate an Error Message.
  • If user enters a mobile number which is not registered with the system, an error message will pop-up.

    • Timer For Resend OTP :

  • Similar to the registration process, a timer is shown on the OTP verification page. If user doesn't enter OTP in the given time frame a "Resend OTP" Link will be displayed.

    • Reset Password Page:

  • Upon verification of OTP, system will redirect user to the "Reset Password"page.
