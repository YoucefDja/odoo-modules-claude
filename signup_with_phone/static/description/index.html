Sign-up using Mobile Number

Now, you can also sign-up using mobile number instead of email-id.

Overview

By default, Odoo allows users to register or sign-up using their email ids, but with this module, users will be able to sign-up using their mobile number.

Features

User can Sign-up with either Email or Mobile.

Upon signing-up a User/Partner with Email/Mobile fields will be created.

Add mobile number with Country-code.

During signing-in one can user the registered mobile number (instead of email-id) and password to access the account.

Setup & Configuration

Version

  • Odoo 15 | 14 | 13 | 12 | 11 | 10 | 9

Configuration

Upon installation of module, necessary configurations will be automatically set like.

Free Account(b2c) will be enabled in Settings -> General Settings -> Custom Account

Employee rights will be assigned if user is of type Portal.

Screenshots

If a user wants to sign-up with mobile number instead of email, we have added a link on signup page under the signup button allowing users to register using mobile number.

Automatic IP Detection: Module will automatically detect system IP and set country code accordingly. However, user can change the country code manually.
Please note that the module is using http://ip-api.com// library to fetch the IP. In case the API doesn't detect the country, US is set as default country.

Country Code Selection: User can select respective country flag from the drop down list for before entering the mobile number if the IP detection fails.

Phone Number Validation is performed.

If and only if the mobile number is valid and all the other details are correct the Sign up button is enable.

User Profile: Upon successful registration, user is created according to the given mobile number and details.

Login With Mobile Link: Similar to the registration process, we have also enabled log-in with mobile number feature for users to acces their account through mobile number.
Under the log-in button user can find the "Log-in with Mobile Number" link.

User needs to enter the mobile number and password and they can easily log-in to their Odoo account.

Reset Password: In case, user forgets the password, he/she will need to reset it through email id.
